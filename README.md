# Phandalin

## Overview

This is my submission for the 2nd round of the gap year. Phandalin is a program where hackers build an RPG video game based on Dungeons & Dragons and receive official Dungeons & Dragons sourcebooks, dice, or any other merchandise for game play. The goal is to get students from all skill levels to design their own original storyline and learn how to build and program a fully playable game from start to finish.

Play the project here: https://anajshah.itch.io/phandalin

## How It Works

I'm not the best at webdev so I ended up using Godot to make the platform. It is supposed to be a game in a website. For the buttons, the user can use their mouse to click them. To see some of the animations, the user can hover over the images. It's a pretty simple website.
