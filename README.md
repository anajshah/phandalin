My submission for 2nd round gap year
Phandalin is a program where hackers build an RPG video game based on Dungeons & Dragons and receive official Dungeons & Dragons sourcebooks, dice, or any other merchandise for game play. The goal is to get students from all skill levels to design their own original storyline and learn how to build and program a fully playable game from start to finish.

It is playable on itch and is not a full blown website
